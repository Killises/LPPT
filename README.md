
# Protección Total – Landing (Next.js + Tailwind + Framer Motion)

Proyecto listo para correr con internacionalización simple (ES/EN), animaciones de entrada y componentes en `/app`.

## Requisitos
- Node.js 18+
- npm o pnpm o yarn

## Instalación
```bash
npm install
npm run dev
```
Abre http://localhost:3000

## Tecnologías
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion

## Estructura
```
/app
  /components
  /data
  globals.css
  layout.jsx
  page.jsx
```

## Personalización
- Edita textos en `app/data/translations.js`.
- Colores y estilos en `app/globals.css` y clases Tailwind.
- Reemplaza placeholders de imágenes por tus assets.
